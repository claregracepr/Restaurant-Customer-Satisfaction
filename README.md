# Restaurant Customer Satisfaction Prediction

## Project Overview
This project predicts restaurant customer satisfaction using machine learning models.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit
- Excel

## Models Used
- Logistic Regression
- KNN
- Random Forest
- Decision Tree

## Key Features
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Model training & evaluation
- Dashboard visualization

## Conclusion
This model helps restaurants improve service quality by identifying key factors affecting customer satisfaction.
